HTML Head

I store info about HTML Head Tags

Title Tag

Title tag provide a title to the page, also helps in the search engine indexing.

Meta Tag

Helps to store meta data related to the webpage, like description, keywords, author, other SEO Stuff also one of the main is the viewport, because it helps to control the look of the device on the smaller screen like mobile phones

Style Tag

It is self explanatory, it provides the functionality to write styling for the webpage internally

Script Tag

It helps to allow javascript functionality to the webpage, also useful or importing external scripts if we further want to add more interactivity

Base Tag

Not used super often, can skip, it provides a base url for all the other links on the page

Link Tag

It defines relationship between the current document and the external document
